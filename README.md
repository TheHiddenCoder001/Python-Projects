# PythonNew

This repository contains several standalone Python automation and scraping projects.
Each folder includes its own script, dependencies, and README.

## Projects

### AmazonScraper
Scrapes Amazon search results by keyword and region, then stores structured output and price history.

### AppMonitor
Monitors running applications, terminates selected targets, and uses tray controls with logging.

### RobloxBadgeDumper
Fetches public Roblox badges for a user and exports results to CSV or SQLite.

### SCPWikiDumper
Downloads SCP Wiki page content and images for a numeric SCP range.

### WebsiteScreenshot
Captures website screenshots on an interval and posts them to a Discord webhook.

## Quick Start

1. Open a project folder.
2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the project script shown in that folder's README.
