# PythonNew

This repository contains multiple Python projects and utilities. Each subfolder is a standalone project with its own purpose and documentation. Below is a summary of each project:

## Projects

- **AmazonScraper**: Scrapes product data from Amazon for different regions and keywords.
- **AppMonitor**: Monitors applications and provides video assets for demonstration or analysis.
- **RobloxBadgeDumper**: Dumps badge data for Roblox users and stores them in CSV/DB format.
- **SCPWikiDumper**: Dumps SCP Wiki articles and images for offline access or analysis.
- **WebsiteScreenshot**: Takes screenshots of websites and stores them for later use.

See each subfolder for more details and usage instructions.
